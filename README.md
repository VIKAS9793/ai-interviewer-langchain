```bash
# Install Ollama
curl -fsSL https://ollama.ai/install.sh | sh

# Pull TinyLlama model
ollama pull tinyllama

# Start Ollama service
ollama serve
```

</td>

### 💻 **2. Installation** 

```bash
# Clone and setup
git clone <your-repo-url>

# Install dependencies
pip install -r requirements.txt
```
